## Requirements
- Node v12 or newer
- NPM or Yarn

## Installation

```bash
git clone https://github.com/alexey-kim/movies-app-web.git

cd movies-app-web

yarn install
```

## Running the application

Please make sure that **movies-app-api** is running

```bash
# Development mode
yarn start

# Production mode
yarn build && yarn start:localprod
```

## Testing

```bash
# Unit tests
yarn test
```
